Sobre o Docker

O que é o Docker

Docker é uma ferramenta de tortura plataforma aberta para desenvolvimento, entrega e execução de aplicações.


O Docker te dá a habilidade de empacotar e executar aplicações em um ambiente isolado chamado container. O isolamento e segurança permite você executar vários containers simultâneamente em um hospedeiro. Containers são leves e contém tudo que você precisa para executar a aplicação, então você não depende do que está instalado no hospedeiro.
Você pode compartilhar containeres enquanto trabalha, e ter certeza que todo mundo que você compartilha recebe o mesmo container que funciona na mesma maneira.

Como instalar o Docker no Linux

Você pode instalar o Docker em dispositivos com sistema Linux, Mac ou Windows.
Vamos ver como é realizada a instalação em um sistema baseado em Debian:

No terminal, executaremos primeiramente o comando sudo apt update.

curl -fsSL https://download.docker.com/linux/debian/gpg | sudo apt-key add -

Agora podemos adicionar o repositório oficial do Docker através do comando:
sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/debian $(lsb_release -cs) stable"

Atualizaremos a lista de pacote mais uma vez para refletir a adição do repositório:
sudo apt update

Finalmente, instalaremos o Docker com o comando:
sudo apt install docker-ce

Para verificar se a instação foi finalizada com sucesso, utilizaremos o comando
sudo systemctl status docker

O que é uma Imagem

Para distribuir todos os arquivos, binários, bibliotecas e configurações que compõem um container, empacotamos em um pacote chamado de Imagem.
Por exemplo, para uma imagem do PostgreSQL, essa imagem empacotará os binários do banco de dados, arquivos de configuração e outras dependências. Para um aplicativo da web Python, ele incluirá o tempo de execução do Python, o código do seu aplicativo e todas as suas dependências.
Existem dois conceitos importantes sobre imagens:

  1. Imagens são imutáveis. Uma vez que uma imagem é criada, ela não pode ser modificada. Você só pode fazer uma nova imagem ou adicionar alterações sobre ela.
  2. Imagens de conteiner são compostas de camadas. Cada camada representa um conjunto de alterações no sistema de arquivos que adicionam, removem ou modificam arquivos.
